# 盛洁熠个人主页

这是一个可直接部署到 GitHub Pages 的静态个人主页，参考 `https://ntdxyg.github.io/` 的学术主页布局制作。

## 文件

- `index.html`：页面结构与中英文内容
- `styles.css`：响应式布局、深浅色主题与卡片样式
- `script.js`：语言切换、主题切换、移动端菜单与滚动动效
- `assets/avatar.jpg`：来自 `头像/主页头像.jpg` 的主页头像

## 本地预览

直接用浏览器打开 `index.html` 即可。也可以在该目录下运行一个静态服务器：

```bash
python3 -m http.server 8000
```
