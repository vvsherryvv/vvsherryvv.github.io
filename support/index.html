Help for release 0.4.4

Support

Wizarding OS is a local-first iPhone app for tasks, projects, saved material, journeys, happy memories, and locally generated plans. It works without an account or server connection.

Getting started

On first launch, choose a visual theme and open the Hall for today’s overview. Use Quick Actions to create a task, project scroll, library item, journey, or memory, or to open the local planning assistant and Recall Light.

Settings includes optional Sample Content for exploring the full app without using personal information.

Tasks and projects

Open Planner to add, edit, schedule, complete, copy, or delete tasks. Filters help separate open and completed work. Project Scrolls hold long-term work, subtasks, progress, and reusable templates.

Memories and photographs

Open Memory Vault to add a happy memory, date, note, tags, happiness level, favorite status, and an optional photograph selected through Apple’s photo picker. Existing memories can be searched, filtered, edited, or deleted.

Memory Magic arranges compatible memories into Timeline, On This Day, Collections, aggregate Insights, and slideshows. Recall Light presents a small selection of guiding memories.

Journeys and travel photos

Use Journey Archive to record a city, country, dates, notes, transport, optional manual coordinates, and multiple local photos. A journey can be reviewed as an album, collage, photo grid, or slideshow. Wizarding OS does not request location permission in this release.

Using Sample Content

In Settings, enable Sample Content and choose to load the Sample Library. It can reset the standard demonstration records or append compatible sample records, depending on the available action. The library covers Hall, Planner, Project Scrolls, Library, Journey Archive, Memory Vault, Memory Magic, Recall Light, and the local assistant.

The exact bundled Sample Content can be removed without deleting unrelated user records.

Exporting and importing JSON backups

Settings can export supported structured records and metadata as JSON. Import uses append-and-deduplicate behavior for supported backup versions, so an item may be skipped when it already exists or cannot be safely read.

Important: user-selected image bytes are not included in JSON backups. Keep original photographs separately. Exported files also leave the app’s control and should be stored or shared carefully.

Removing local content

Delete individual tasks, projects, saved items, journeys, memories, or collections inside the app. Settings can remove the exact bundled Sample Content. Deleting the app removes its local container, subject to device backups managed by the user or Apple. Export a backup first when records may be needed later.

Accessibility

Wizarding OS supports Dynamic Type, VoiceOver labels, Reduce Motion, and accessibility-aware slideshow controls. If text is truncated, increase available screen space, try portrait orientation, or use the device’s text-size and display settings. Reduce Motion limits non-essential effects in Recall Light and slideshows.

Offline behavior

Core planning, memory, travel, Sample Content, backup, and local-assistant features work offline. Photos already selected into the app remain local. Operating-system sharing, App Store services, and opening external map destinations may require a network connection.

Troubleshooting

App does not launch

Confirm the device runs a supported iOS version and has free storage, then restart the app and device. Avoid deleting the app before securing any backup you need.

Photo does not appear

Retry selection through the system photo picker, allow processing to finish, and confirm free storage is available. JSON imports restore photo metadata but not user-selected image bytes.

Import skips items

Confirm the file came from a supported Wizarding OS backup version. Existing or duplicate records may be skipped by design. The import summary reports what was accepted or skipped.

Sample Content is missing

Open Settings, enable Sample Content, and load the Sample Library. The visible upgrade action can prepare recognized older demonstration records without rewriting user-created memories.

Upgrade from 0.4.3

Install 0.4.4 over 0.4.3 using the same formal app identity. The app performs a lightweight local data migration and upgrades only precisely recognized demonstration records. Back up supported structured data first. Builds installed under an older placeholder bundle identifier use a different app container and do not migrate automatically.

Privacy

Do not send private memory text, photographs, exported backups, credentials, signing identities, or provisioning information in a support request. Read the Wizarding OS Privacy Policy for details.

Contact support

When the monitored support address is published, include the app version, iOS version, device model, the action that failed, and non-sensitive reproduction steps. This site intentionally has no contact form.
